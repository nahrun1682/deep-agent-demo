# Common Blackboard Skill

## SKILL

Use this skill whenever you work inside the study demo.

- Keep the blackboard files readable and consistent.
- Treat structured output as canonical state and Markdown as a projection.
- Record MCP usage with reason and summary.
- Record memory proposals before promoting any long-term memory.
- Prefer trace entries with actor, action, target, reason, result, and next step.
